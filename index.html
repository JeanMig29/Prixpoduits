Open Prices API

Prix produits monde en direct

Interface HTML autonome basee uniquement sur l'API Open Prices. Elle affiche un flux recent de prix publics et permet de rechercher un produit par code-barres EAN/UPC, sans installation et donc publiable tel quel sur GitHub Pages.

Mode recommande: l'appareil photo natif du telephone. Le scan dans la page reste disponible, mais la mise au point web depend fortement du navigateur et du webview utilise.

Ouvrez cette page en HTTPS sur votre telephone, autorisez la camera, puis cadrez le code-barres.
